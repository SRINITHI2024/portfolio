PROJECT

  • LEAF DISEASE DETECTION USING ML

    • Leaf disease detection using machine learning is a project focused on identifying and diagnosing plant diseases through the analysis of leaf images. This project involves the application of computer vision techniques and machine learning algorithms to classify and recognize various plant diseases, which can be crucial for agriculture and gardening.

  • WASTE MANAGEMENT USING ML

    • Sanitary pad waste management using machine learning is a project aimed at addressing the environmental and health challenges posed by the disposal of sanitary pads. This project seeks to leverage machine learning techniques to optimize the collection, segregation, and processing of sanitary pad waste, contributing to more sustainable waste management practices.
